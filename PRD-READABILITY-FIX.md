# PRD: 녹색 배경 텍스트 가독성 개선

## 문서 정보
- **작성일**: 2025-12-18
- **프로젝트**: 은인자금파트너스 홈페이지
- **버전**: 1.0

---

## 1. 개요

### 1.1 배경
은인자금파트너스 홈페이지의 브랜드 컬러를 네이비에서 포레스트 그린으로 변경하면서, 녹색 오버레이/배경 위에 녹색 계열 텍스트가 사용되어 가독성이 저하되는 문제가 발생했습니다.

### 1.2 목표
녹색 배경 위의 강조 텍스트를 흰색 계열로 변경하여 시각적 대비를 개선하고 가독성을 확보합니다.

## 4. 색상 가이드라인

### 4.1 녹색 배경 (오버레이) 위에서 사용할 색상
```css
/* 강조 텍스트 - 권장 */
color: #FFFFFF;
text-shadow: 0 0 15px rgba(255, 255, 255, 0.6);

/* 그라데이션 강조 - 권장 */
background: linear-gradient(45deg, #FFFFFF, #E8F5E9, #FFFFFF);
```

### 4.2 밝은 배경 위에서 사용할 색상
```css
/* 강조 텍스트 - 기존 유지 */
color: #74C69D; /* 또는 #52B788 */
```

### 4.3 배경 색상 참고
- 녹색 오버레이: `rgba(45, 106, 79, 0.85)` ~ `rgba(27, 67, 50, 0.75)`
- 밝은 녹색 배경: `#D8F3DC` (primary-light)
- 흰색/회색 배경: `#FFFFFF`, `#F9FAFB` (neutral-50)
